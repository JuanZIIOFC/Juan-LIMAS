# Projeto – Dashboard Cotações BCB

##  Objetivo
Consumir a API do Banco Central do Brasil para obter cotações de moedas e construir um dashboard interativo no Power BI.

##  Tecnologias utilizadas
- Python
- requests
- Power BI

##  Estrutura do projeto
- coleta_moedas_bcb.py → Script responsável pela coleta dos dados da API do BCB.
- Dashboard_Cotacoes_BCB.pbix → Dashboard desenvolvido no Power BI.

##  Como executar
1. Instalar as dependências:
   pip install requests
2. Executar o script:
   python coleta_moedas_bcb.py
